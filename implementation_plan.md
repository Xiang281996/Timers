# 腕錶詢價網頁小工具 (Timers)

根據您的需求，我們將開發一個名為「Timers」的腕錶詢價小工具。這個專案會著重於打造一個具備高級質感、互動性強且符合腕錶收藏家品味的網頁介面。

## 系統架構

我們將使用純 HTML, CSS (Vanilla CSS), 與 JavaScript 來進行開發，並使用 Vite 作為開發伺服器進行快速測試。

## Proposed Changes

### UI / UX 設計與實作
將包含以下各區塊的樣式與互動邏輯：

1. **頂部標誌與主視覺區**
    - 專屬設計的 SVG Logo：以金、銀、黃、橘色調設計，結合「時間」、「齒輪」與「Timers」字樣。
    - 營造專業、沉穩的網站風格 (例如深色模式搭配低調奢華的金屬光澤)。

2. **搜尋表單區**
    - 品牌下拉選單：列出知名腕錶品牌（如 Rolex, Omega, Patek Philippe, Audemars Piguet 等），以及「其他」選項（非必填）。
    - 搜尋輸入框：讓使用者打上腕錶名稱或型號（這是必填欄位）。
    - 精緻的查詢按鈕。

3. **確認錶款區塊 (動態切換)**
    - 送出查詢後，先隱藏表單區，顯示「初步搜尋到的腕表圖片」。
    - 詢問互動：「這是否符合您所選的腕錶？」
    - 提供【是】與【否】按鈕。
    - **若選擇【否】**：提示是否「重新搜尋」或「停止搜尋」。
    - **若選擇【是】**：進入詳細結果區塊。

4. **結果顯示區塊 (動態切換)**
    - 以表格格式詳盡展示腕錶資訊：腕錶品牌、腕錶名稱、型號、官網價格、市場價格、錶徑大小(mm)、錶耳長度、腕錶材質、錶帶材質、動力儲存(小時)、功能。若無資料將填入 `-`。
    - 表格下方以多圖列展現腕錶（包含正面、側面、背面照或機芯透背圖）。

5. **模擬資料流程 (Mock Data)**
    - 為了讓網頁小工具的功能完整可見，我會實作一組模擬流程（例如搜尋「Submariner」則跳出勞力士水鬼的資料與圖片）來展示 AI 互動的體驗。

## User Review Required

> [!IMPORTANT]
> 1. 目前此小工具主要是前端介面的完整實作。背後搜尋腕錶資訊的功能將會先使用「模擬資料 (Mock Data)」供您測試與體驗整體的點擊與顯示流程（後續您再依需要串接後端AI或爬蟲）。這樣的安排是否符合預期？
> 2. 外觀設計上，預計採用深色背景（Dark mode）搭配金色/銀色的字體與邊框來呈現「專業銷售顧問」及「高級腕錶」的質感，請問是否可以？

## Verification Plan

### Manual Verification
- 啟動 local 開發伺服器，並提供預覽網址。
- 實際操作搜尋流程：輸入任意錶款 -> 顯示圖案確認 -> 點擊「是」 -> 看到表格與詳細照片。
- 檢查 SVG Logo 和網站介面的顏色與設計是否達到高級與專業的質感。
